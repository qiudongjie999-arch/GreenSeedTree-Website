# 青稷树 Logo - AI生成工具操作指南

---

## 推荐工具

### 首选：标小智（中文，免费）

**网址**：https://www.logosc.cn

**优点**：
- 中文界面，操作简单
- 免费生成，可下载PNG
- 支持中文品牌名

**操作步骤**：

1. 打开 https://www.logosc.cn
2. 输入品牌名：`青稷树`
3. 选择行业：`科技` 或 `互联网`
4. 点击"开始设计"
5. 系统会生成多个Logo方案
6. 选择喜欢的方案，可以微调：
   - 颜色：选择绿色系
   - 图标：选择"植物"、"种子"、"树"相关
   - 字体：选择现代简约风格
7. 下载PNG文件（免费）

---

### 备选一：Looka（国际，付费）

**网址**：https://looka.com

**优点**：
- 设计质量高
- 提供品牌全套方案
- 可定制细节

**操作步骤**：

1. 打开 https://looka.com
2. 点击"Start your logo"
3. 输入品牌名：`GreenSeedTree`
4. 选择行业：`Technology`
5. 选择喜欢的Logo风格（3-5个）
6. 选择颜色：Green
7. 选择图标风格：Nature, Growth, Minimal
8. 系统生成方案后，选择喜欢的
9. 购买套餐下载（约$20）

### 备选二：Brandmark（高质量）

**网址**：https://brandmark.io

**优点**：
- AI设计质量最高

- 配色方案专业

### 备选三：Canva（免费模板）

**网址**：https://www.canva.com

**优点**：
- 完全免费
- 可完全自定义
- 中文界面

**操作步骤**：

1. 打开 https://www.canva.com
2. 搜索"Logo设计"
3. 选择一个简约的Logo模板
4. 修改文字为"青稷树"
5. 搜索元素"种子"、"发芽"、"树"
6. 添加并调整元素
7. 修改颜色为绿色
8. 下载PNG文件

## 生成时的关键词

如果工具支持输入描述，使用以下关键词：

**中文**：
```
种子发芽，两片嫩叶向上生长，简约现代，
科技感，绿色渐变，AI公司，成长，希望
```

**英文**：
```
seed sprouting, two leaves growing upward,
minimalist modern, tech style, gradient green,
AI company, growth, hope, clean lines
```

## 生成后检查清单

下载Logo后，检查以下几点：

- [ ] 图形是否清晰，无锯齿
- [ ] 颜色是否为绿色系
- [ ] 图形是否简洁，易于识别
- [ ] 是否有"种子发芽"的意象
- [ ] 放大缩小后是否依然清晰
- [ ] 在白色背景上是否清晰
- [ ] 在深色背景上是否清晰

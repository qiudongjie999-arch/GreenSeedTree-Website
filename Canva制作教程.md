# 青稷树 Logo - Canva 制作教程

---

## 一、准备工作

### 打开Canva

**网址**：https://www.canva.com

**如果没有账号**：
1. 点击"注册"
2. 可以用微信、邮箱或手机号注册
3. 选择免费计划即可

---

## 二、创建Logo项目

### 方法一：从空白开始

1. 登录Canva后，点击右上角"创建设计"
2. 搜索"Logo"或选择"自定义尺寸"
3. 输入尺寸：`1000 × 1000` 像素（正方形）
4. 点击"创建新设计"

### 方法二：使用Logo模板

1. 登录Canva后，在搜索框输入"Logo"
2. 浏览模板，选择一个简约的模板
3. 点击使用，然后删除不需要的元素

---

## 三、制作"种子发芽"Logo

### 步骤1：设置背景

1. 点击左侧"背景"
2. 选择白色背景（或透明背景，方便后续使用）

### 步骤2：添加种子壳（底部椭圆）

1. 点击左侧"元素"
2. 搜索"椭圆"或"圆形"
3. 选择一个实心椭圆
4. 拖拽到画布中央底部
5. 调整大小和位置
6. 点击颜色，选择绿色：`#16A34A`
7. 可以添加渐变效果：
   - 点击"效果" → "渐变"
   - 设置渐变色：从 `#14532D` 到 `#16A34A`

### 步骤3：添加茎（中间竖线）

1. 点击左侧"元素"
2. 搜索"线条"或"矩形"
3. 选择一个细长的矩形或线条
4. 拖拽到种子壳上方中央
5. 调整宽度：约 10-20 像素
6. 调整高度：约 200-300 像素
7. 设置颜色：`#22C55E`

### 步骤4：添加嫩叶（顶部两片叶子）

**方法A：使用现成的叶子元素**

1. 点击左侧"元素"
2. 搜索"叶子"或"leaf"
3. 选择一个简约的叶子形状
4. 复制一份（Ctrl+C，Ctrl+V）
5. 将两片叶子对称放置在茎的两侧
6. 调整角度，微微向外展开
7. 设置颜色：`#4ADE80`

**方法B：用形状自己画**

1. 点击左侧"元素"
2. 搜索"三角形"
3. 选择一个三角形
4. 点击"效果" → "圆角"（让边缘圆润）
5. 旋转三角形，使其尖端朝上
6. 复制一份，水平翻转
7. 对称放置在茎的两侧
8. 设置颜色：`#4ADE80`

### 步骤5：添加根基（可选）

1. 点击左侧"元素"
2. 搜索"曲线"或"弧形"
3. 在种子壳底部添加一个小弧形
4. 设置颜色：`#14532D`（深绿）

### 步骤6：调整整体

1. 选中所有元素（Ctrl+A）
2. 点击"位置" → "居中"
3. 调整各元素的比例和位置
4. 确保整体协调

---

## 四、添加品牌名（可选）

### 横式组合

1. 点击左侧"文字"
2. 添加标题：`青稷树`
3. 选择字体：
   - 思源黑体（简约现代）
   - 方正兰亭黑（商务专业）
   - 站酷高端黑（设计感）
4. 设置字号：与Logo图形协调
5. 设置颜色：`#1E293B`（深灰）或 `#16A34A`（绿色）
6. 在品牌名下方添加英文：`GreenSeedTree`
7. 设置较小字号，颜色稍浅

### 竖式组合

1. 将Logo图形放在上方
2. 品牌名放在下方
3. 整体居中对齐

## 五、配色参考

### 推荐配色

| 元素 | 颜色 | 色值 |
|------|------|------|
| 嫩叶 | 浅绿 | #4ADE80 |
| 茎 | 活力绿 | #22C55E |
| 种子壳 | 科技绿 | #16A34A |
| 根基 | 深绿 | #14532D |
| 品牌名 | 深灰 | #1E293B |

### 渐变效果

如果想让Logo更有层次感：

1. 选中元素
2. 点击"效果" → "渐变"
3. 设置渐变色：
   - 嫩叶：`#22C55E` → `#4ADE80`
   - 种子壳：`#14532D` → `#16A34A`

## 六、导出文件

### 导出PNG（推荐）

1. 点击右上角"分享"
2. 选择"下载"
3. 文件类型：PNG
4. 勾选"透明背景"（重要！）
5. 点击"下载"

### 导出不同尺寸

1. 下载后，可以再次调整画布尺寸
2. 导出不同尺寸的版本：
   - 网站用：500×500px
   - 社交媒体：400×400px
   - Favicon：32×32px

## 九、如果遇到问题

### 问题1：找不到合适的叶子元素

**解决**：
- 搜索关键词尝试：`叶子`、`leaf`、`植物`、`nature`
- 或者用三角形+圆角自己画

### 问题2：颜色不匹配

**解决**：
- 点击颜色 → 选择"+"号
- 输入具体色值：`#16A34A`

### 问题3：导出后有白底

**解决**：
- 下载时勾选"透明背景"
- 如果没有这个选项，说明使用了不支持透明的元素
- 检查是否使用了照片或某些效果

### 问题4：想要更专业的设计

**解决**：
- 可以在Canva中完成基础设计
- 然后找设计师进行精修
- 或者使用Figma进行更精细的调整

# 青稷树 Logo 设计规范 - 方案一：种子发芽

---

## 一、Logo设计说明

### 核心图形

**种子发芽**：一颗种子正在发芽，两片嫩叶向上生长

### 设计元素拆解

```
         ╭─────╮
        ╱   🌱   ╲      ← 嫩叶（两片）
       │    │    │
       │    │    │      ← 茎
        ╲   │   ╱
         ╰──┴──╯        ← 种子壳
            │
         ╭──┴──╮
        ╱       ╲       ← 根基
       ╰─────────╯
```

### 图形比例

- **整体比例**：1:1（正方形）或 3:4（竖长）
- **叶子部分**：占整体高度的 40%
- **茎部分**：占整体高度的 30%
- **根基部分**：占整体高度的 30%

### 设计细节

**嫩叶**：
- 两片对称的叶子
- 形状：圆角三角形，微微向外展开
- 角度：与中心线成 15-20° 角
- 边缘：圆润，无尖角

**茎**：
- 一条细长的竖线
- 从种子壳中心向上延伸
- 连接种子壳和嫩叶

**种子壳**：
- 椭圆形或水滴形
- 包裹茎的底部
- 象征"稷"（种子）

**根基**：
- 底部略微展开
- 象征稳健、可靠

## 二、配色方案

### 主配色

**渐变绿**（从下到上）：
```
根基：#14532D（深绿）
种子壳：#16A34A（科技绿）
茎：#22C55E（活力绿）
嫩叶：#4ADE80（浅绿）
```

### 单色版本

**科技绿**：#16A34A
- 用于印刷、单色场景
- 确保在黑白打印时清晰可辨

### 反白版本

**白色**：#FFFFFF
- 用于深色背景
- 保持图形清晰度

## 四、应用场景规范

### 网站Header

```
高度：40-60px
Logo位置：左侧
品牌名：Logo右侧，字号与Logo高度协调
```

### Favicon

```
尺寸：32×32px / 16×16px
内容：仅保留嫩叶+茎的简化图形
颜色：#16A34A
```

### 名片

```
正面：
- Logo位置：左上角或居中
- 尺寸：15-20mm高度

背面：
- Logo居中放大
- 尺寸：30-40mm高度
- 下方配Slogan
```

### 社交媒体头像

```
尺寸：400×400px
内容：Logo居中
背景：白色或#F0FDF4（浅绿）
```

## 五、Logo制作方法

### 方法一：专业设计师（推荐）

**优点**：
- 原创性强
- 可定制细节
- 提供完整文件

**平台推荐**：
- 猪八戒网：www.zbj.com
- 一品威客：www.epwk.com
- 站酷：www.zcool.com.cn
- Fiverr（国际）：www.fiverr.com

**预算参考**：
- 基础设计：500-1500元
- 专业设计：1500-5000元
- 高端定制：5000元以上

**交付文件**：
- AI/EPS 矢量源文件
- PNG 透明底（多尺寸）
- SVG 网页用
- PDF 印刷用

---

### 方法三：自己用设计软件制作

**推荐软件**：
- **Figma**（免费）：在线协作设计工具
- **Illustrator**（付费）：专业矢量设计
- **Inkscape**（免费）：开源矢量设计

**制作步骤**：
1. 创建画布（建议 1000×1000px）
2. 用形状工具绘制：
   - 椭圆工具：种子壳
   - 三角形工具：嫩叶（圆角处理）
   - 线条工具：茎
3. 应用渐变色
4. 导出多格式文件

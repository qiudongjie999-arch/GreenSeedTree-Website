# GreenSeedTree 网站上线指南

## 当前状态
- 域名：GreenSeedTree.com（腾讯云）
- 备案类型：企业备案
- 腾讯云：已实名认证
- 目标：双线部署（Vercel快速上线 + 腾讯云COS长期托管）

---

## 方案A：Vercel快速上线（今天就能访问）

### 第一步：注册GitHub账号（5分钟）

1. 打开 https://github.com
2. 点击右上角 "Sign up"
3. 填写邮箱、密码、用户名
4. 完成邮箱验证
5. 选择免费计划（Free）

**用户名建议**：greenseedtree 或 greenseedtree-official

### 第二步：创建GitHub仓库（3分钟）

1. 登录GitHub后，点击右上角 "+" → "New repository"
2. 仓库名称填写：`greenseedtree.com` 或 `greenseedtree-website`
3. 选择 "Public"（公开）
4. 勾选 "Add a README file"
5. 点击 "Create repository"

### 第三步：上传网站文件（5分钟）

**方法一：网页上传（简单）**
1. 进入刚创建的仓库
2. 点击 "Add file" → "Upload files"
3. 将 `index.html` 文件拖拽上传
4. 点击 "Commit changes"

**方法二：使用GitHub Desktop（推荐，后续更新方便）**
1. 下载安装 GitHub Desktop：https://desktop.github.com
2. 登录你的GitHub账号
3. Clone仓库到本地
4. 将 `index.html` 复制到仓库文件夹
5. 提交并推送

### 第四步：注册Vercel账号（3分钟）

1. 打开 https://vercel.com
2. 点击 "Sign Up"
3. 选择 "Continue with GitHub"
4. 授权Vercel访问你的GitHub

### 第五步：部署网站（2分钟）

1. 登录Vercel后，点击 "Add New..." → "Project"
2. 选择你刚创建的GitHub仓库
3. 点击 "Import"
4. 项目名称保持默认或改为 `greenseedtree`
5. 点击 "Deploy"
6. 等待1-2分钟，部署完成

**部署成功后**，你会获得一个临时域名，类似：
`greenseedtree.vercel.app`

此时网站已经上线，可以通过这个地址访问！

### 第六步：绑定自定义域名（5分钟）

1. 在Vercel项目页面，点击 "Settings" → "Domains"
2. 输入 `greenseedtree.com`，点击 "Add"
3. 再添加 `www.greenseedtree.com`
4. Vercel会显示DNS配置信息

**在腾讯云配置DNS解析**：
1. 登录腾讯云控制台
2. 进入 "DNS解析DNSPod"
3. 找到 greenseedtree.com 域名
4. 添加记录：
   - 主机记录：`@`
   - 记录类型：CNAME
   - 记录值：`cname.vercel-dns.com`
   - TTL：默认
5. 再添加：
   - 主机记录：`www`
   - 记录类型：CNAME
   - 记录值：`cname.vercel-dns.com`
   - TTL：默认

**等待DNS生效**：通常5-30分钟

### 第七步：配置SSL证书（自动）

Vercel会自动为你的域名配置免费SSL证书，无需额外操作。

## 方案B：腾讯云COS部署（备案通过后使用）

### 第一步：企业备案（10-20个工作日）

**所需材料**：
1. 营业执照照片（清晰、完整）
2. 法人身份证正反面照片
3. 网站负责人身份证正反面照片（可以和法人相同）
4. 网站负责人手持身份证照片
5. 网站负责人核验单（腾讯云会提供模板，打印后签字拍照）

**备案流程**：
1. 登录腾讯云控制台
2. 搜索 "网站备案"，进入备案系统
3. 选择 "企业备案"
4. 填写主体信息：
   - 企业名称（与营业执照一致）
   - 统一社会信用代码
   - 企业注册地址
   - 法人信息
5. 填写网站信息：
   - 网站名称：建议用 "GreenSeedTree官网" 或 "绿籽树科技官网"
   - 网站域名：greenseedtree.com
   - 网站服务内容：选择 "其他"
   - 网站负责人信息
6. 上传材料：
   - 营业执照
   - 身份证照片
   - 手持身份证照片
   - 核验单
7. 提交审核

**注意事项**：
- 网站名称不能用"科技"、"网络"、"电子商务"等词（除非有相关资质）
- 建议用品牌名或公司简称
- 备案期间网站不能访问（所以先用Vercel）

### 第二步：开通COS服务（备案通过后）

1. 登录腾讯云控制台
2. 搜索 "对象存储COS"
3. 点击 "立即使用"
4. 创建存储桶：
   - 名称：greenseedtree（自定义）
   - 地域：选择离你用户最近的（如广州、上海）
   - 访问权限：公有读、私有写
   - 版本控制：不开启
5. 进入存储桶，开启 "静态网站" 功能
6. 上传 `index.html` 文件

### 第三步：配置CDN加速

1. 在COS存储桶设置中，找到 "域名与传输管理"
2. 点击 "自定义CDN加速域名"
3. 添加域名：`greenseedtree.com`
4. 源站类型：默认源站
5. 提交后，获取CNAME地址

### 第四步：配置SSL证书

1. 腾讯云控制台 → SSL证书
2. 申请免费证书：
   - 证书类型：DV SSL证书（免费）
   - 域名：greenseedtree.com
   - 验证方式：DNS验证
3. 按提示在DNS解析中添加验证记录
4. 等待验证通过（通常几小时）
5. 下载证书，配置到CDN

### 第五步：修改DNS解析

将之前指向Vercel的DNS记录，改为指向腾讯云CDN的CNAME地址。

## 后续维护

### 更新网站内容

**方法一：修改GitHub仓库**
1. 修改本地 `index.html` 文件
2. 上传到GitHub仓库
3. Vercel会自动重新部署（1-2分钟）

**方法二：直接修改腾讯云COS**
1. 登录腾讯云控制台
2. 进入COS存储桶
3. 上传新的 `index.html` 覆盖旧文件

### 添加表单后端

如果需要收集用户提交的表单，可以：
1. 使用腾讯云云函数（SCF）
2. 使用第三方服务：飞书表单、Typeform、Formspree等
3. 接入企业微信/飞书机器人通知

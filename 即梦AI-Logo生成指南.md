## 即梦AI操作步骤

### 第一步：访问即梦AI

1. 打开浏览器，访问：https://jimeng.jianying.com/
2. 登录账号（支持抖音、手机号登录）

### 第二步：创建图片

1. 点击"AI作画"或"文生图"
2. 选择模型：推荐"通用"或"设计"模型
3. 输入Prompt（复制上面的Prompt）
4. 设置参数：
   - **图片尺寸**：1:1（正方形）
   - **生成数量**：4张（默认）
   - **风格强度**：中等

### 第三步：生成与选择

1. 点击"生成"
2. 等待生成完成（约30秒）
3. 查看4张生成的图片
4. 选择最满意的版本

### 第四步：下载与优化

1. 点击满意的图片
2. 选择"下载高清图"
3. 使用去背景工具去除背景
4. 根据需要调整颜色

## 高级技巧

### 技巧1：添加负面词

在Prompt后添加负面词，避免不想要的元素：

```
极简Logo设计，种子发芽...（正面Prompt）

负面词：文字、阴影、渐变、3D效果、复杂、模糊、低质量、不对称
```

### 技巧2：多次迭代

1. 第一次生成后，选择最接近的版本
2. 使用"以此为参考"功能
3. 微调Prompt，再次生成
4. 重复直到满意

### 技巧3：风格参考

如果即梦AI支持上传参考图：
1. 上传喜欢的Logo风格图片
2. 让AI参考该风格生成
3. 更容易得到理想效果

## 后期处理

### 必要处理

1. **去背景**
   - 工具：remove.bg、Adobe Express、美图秀秀
   - 确保背景完全透明

2. **颜色校正**
   - 使用Photoshop或Figma
   - 调整为品牌标准色：#16A34A、#22C55E、#4ADE80

3. **矢量化**
   - 工具：Vectorizer.ai、Adobe Illustrator
   - 转换为SVG格式，便于缩放

4. **边缘优化**
   - 清理锯齿
   - 确保边缘平滑

### 推荐工具

| 用途 | 工具 | 链接 |
|------|------|------|
| 去背景 | remove.bg | https://www.remove.bg/ |
| 去背景 | Adobe Express | https://www.adobe.com/express/feature/remove-background |
| 矢量化 | Vectorizer.ai | https://vectorizer.ai/ |
| 图片编辑 | Photopea | https://www.photopea.com/ |
| 格式转换 | CloudConvert | https://cloudconvert.com/ |
